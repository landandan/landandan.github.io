Hexo leanCloud阅读统计配置

发表于 2018-03-21 | 分类于 随笔 ， Hexo | 热度: ℃

昨天更换了最新的Next 6.0.6主题，官方网站还没有上线，当时就找了篇个人的教程配置了下，发现有点小问题。
今天又重新研究了下。

第一个问题是发现6.0.6版的lib文件夹下缺了不少文件，导致很多效果无效，所幸老版的lib下都有，clone了一份就解决了。

第二个问题是leanCloud阅读统计，当时看的文章可能没写清楚，下面我再写一遍。

  1. 注册/登录leanCloud账户
  2. 创建应用

  3. 进入应用Key，复制App ID及App Key，并将它们填到next主题_config.yml下的leancloud_visitors

  4. 进入存储创建Class
