快应用初探

发表于 2018-03-22 | 热度: ℃

快应用官网
快应用官方文档

官方文档写的还是比较详细的
遇到的问题都有提到解决

搭建环境

npm install -g hap-toolkit

创建项目

hap init XXXX(项目名称)
npm install

编译项目

npm run build

npm run build的时候报错


执行hap update --force再npm run build就没问题了

自动重新编译

npm run watch

安装快应用调试器

遇到了如下这种情况

需要安装下平台预览版，再打开就正常了

启动HTTP服务器

npm run server

服务启动起来会生成二维码，用调试器直接扫描二维码即可预览（同一wifi下）

调试

点击开始调试会打开chrome devtools

总结

整个流程还是非常简洁的，手机上的运行也是非常流畅
