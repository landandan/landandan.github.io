GitHub上fork后的项目如何同步原项目的更新

发表于 2017-05-26 | 分类于 Git ， GitHub | 热度: ℃

  1. 给fork配置远程仓库

    • 使用git remote -v查看远程状态
    • 添加一个将被同步的上游仓库
      git remote add upstream 上游仓库地址
    • git remote -v查看是否添加成功

  2. 同步给fork

    • 从上游仓库fetch，并会被存储在本地分支upstream/master
      git fetch upstream
    • 切换到本地master分支（如果不在的话）
      git checkout master
    • 把upstream/master分支合并到本地master
      git merge upstrea/master
    • 如何需要同步到GitHub上的话，执行以下代码即可
      git push origin master
