如果环境多个，并有可能需要配置不同参数，这样就需要添加很多个配置。显得不够灵活，那我们为什么不直接在npm run的时候添加参数呢？
node命令的入参获取本质上就是用process.argv获取。但是我们直接在npm run serve后面添加参数vue.config.js中process.argv获取不到入参，也就是说后面加的参数都无效。
